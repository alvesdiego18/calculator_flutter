## Running this app

1 - Install [Flutter](https://flutter.dev/) from this [doc](https://flutter.dev/docs/get-started/install)

2 - Clone this repository to your local machine

3 - Run app
```
flutter run --no-sound-null-safety
```

4 - Enjoy
